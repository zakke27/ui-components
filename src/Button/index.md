# Button 按钮

### 按钮类型

按钮分为五种类型`type`，默认是`default`。

```tsx
import React from 'react';
import { Button } from 'dr-ui-components';

export default () => {
  return (
    <>
      <Button type="primary">primary</Button>
      <Button type="primary">primary31ddkasldakdlqmwerq;we</Button>
      <Button>secondary</Button>
    </>
  );
};
```
